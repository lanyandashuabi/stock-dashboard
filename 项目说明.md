# 光明宗 · 股票研究看板

> A股/港股个性化股票研究看板 | v2.4.1

---

## 项目概述

- **名称**: 光明宗 · 股票研究看板
- **用途**: 个人 A 股/港股投资复盘与研究
- **市场**: A 股（上证/深证/北交所）+ 港股

## 技术栈

| 层 | 技术 |
|----|------|
| 前端 | React 18 + TypeScript + Vite + Tailwind CSS |
| 后端 | Express 5 + TypeScript |
| 数据库 | SQLite (better-sqlite3) |
| 图表 | ECharts |
| 进程管理 | PM2 / supervisord |

## 核心功能

1. **宏观数据** — 7大指数、市场情绪、宏观经济、货币政策、全球市场、行业轮动与资金方向
2. **行业分析** — AI算力/机器人/创新药/油运 四大行业产业链 + 龙头追踪
3. **个股池** — SQLite 持久化、行业筛选、添加/删除、K线弹窗
4. **观察清单** — 表格展示、备注编辑、预警价
5. **K线图** — ECharts 蜡烛图 + MA5/MA10/MA20 + MACD + 成交量
6. **研报智能分析** — 粘贴研报内容，自动提取个股/行业/风险，一键入池
7. **定时任务** — 每个工作日 15:00 自动抓取收盘数据

## 数据源

| 数据 | 来源 |
|------|------|
| A股实时行情 | 新浪财经 hq.sinajs.cn |

| A股日K | 新浪财经 K线接口 |
| 港股日K | 腾讯财经 web.ifzq.gtimg.cn |
