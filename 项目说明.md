## 项目概述

- **名称**: 光明宗 · 股票研究看板
- **用途**: 个人 A 股/港股投资复盘与研究
- **市场**: A 股（上证/深证/北交所）+ 港股
- **配色**: 红白金融风格（#c41e3a 中国红 + #fafafa 暖白）
- **Logo**: 光明宗八角星 SVG

## 技术栈

| 层 | 技术 |
|----|------|
| 前端 | React 18 + TypeScript + Vite + Tailwind CSS |
| 后端 | Express 5 + TypeScript |
| 数据库 | SQLite (better-sqlite3) |
| 图表 | ECharts |
| 进程管理 | PM2 / supervisord |

## 核心功能

1. **宏观数据** — 7大指数、市场情绪、宏观经济、货币政策、全球市场、行业轮动与资金方向
2. **行业分析** — AI算力/机器人/创新药/油运 四大行业产业链 + 龙头追踪
3. **个股池** — SQLite 持久化、行业筛选、添加/删除、K线弹窗
4. **观察清单** — 表格展示、备注编辑、预警价
5. **K线图** — ECharts 蜡烛图 + MA5/MA10/MA20 + MACD + 成交量
6. **研报智能分析** — 粘贴研报内容，自动提取个股/行业/风险，一键入池（规则引擎：40+ 股票映射 + 8 大行业关键词）
7. **定时任务** — 每个工作日 15:00 自动抓取收盘数据

## 数据源

| 数据 | 来源 | 编码 |
|------|------|------|
| A股实时行情 | 新浪财经 hq.sinajs.cn | GBK |
| 港股实时行情 | 新浪财经 hq.sinajs.cn | GBK |

| 港股日K | 腾讯财经 web.ifzq.gtimg.cn | UTF-8 |

| 低频宏观 | 静态数据（标注来源） | - |

## 已知问题与注意事项

- 数据源为免费接口（新浪/腾讯），非官方 API，稳定性有限
- 港股字段数（~18）少于 A 股（~30+），代码中已做区分处理
- 沙箱环境可能需要配置 GitHub token 才能 clone
- 首次启动会自动创建 SQLite 数据库
